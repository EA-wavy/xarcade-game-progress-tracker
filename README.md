# XArcade — Game Progress Tracker
A browser-based game progress tracker that allows users to create a personal game catalogue, track their completion progress, and organise their library by status such as currently playing, completed, or backlog. It runs in the browser for simple and universal access. To run locally click on the index file 


## Features

- Add games with a title, genre, status and completion percentage
- Track games across three states: Playing, Completed and Backlog
- Edit or delete any game entry inline
- Filter by status and sort by title or date added
- Games grouped by genre in the My Games view
- Dashboard showing your five most recently added games
- Clean dark-themed UI — no installation or login required

## How to Run
1. Clone or download this repository
2. Open the project folder in VS Code
3. Right-click on the "Go Live" button at the bottom right of the screen
4. The app will open in your browser automatically
